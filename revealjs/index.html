Tipos de datos

Datos Primitivos

Un tipo de datos es un conjunto de valores y un conjunto de operaciones definidas en ellos. Por ejemplo, estamos familiarizados con los números y con las operaciones definidas en ellos, tales como la suma y la multiplicación. Existen ocho tipos diferentes de datos incorporados en Java, en su mayoría diferentes tipos de números. Utilizamos el tipo de sistema para cadenas de caractere (String) con tanta frecuencia que también lo consideramos aquí.

Nombre Tipo

Enteros

Consiste en un numero entero entre −231 and 231 − 1   (−2,147,483,648 to 2,147,483,647). 


							int a,b,c;
							a= 10;
							b= -345;
							c=456;
						

							int a = 10;
							int b = -345;
							int c = 456;

Caracteres

Los caracteres son simbolos alfanumericos. Generalmente se utilizan para asignar valores. 


							char ch = 'a';
							char ab;
							ab = 'b'
						

							String a, b, c;
							a = "Hello,";
							b = " Bob";
							c = a + b;

Decimal/Flotante

El tipo double es para representar números flotante, por ejemplo, para uso en aplicaciones científicas. La representación interna es como la notación científica, por lo que podemos calcular con números reales en un rango enorme. Podemos especificar un número flotante usando una cadena de dígitos con un punto decimal, por ejemplo, 3.14159 para una aproximación de seis dígitos a la constante matemática pi. 


							double a = 5.68634567
							float b = 6.34

Booleanos

Los tipos booleanos solo tienen 2 valores true o false 


							boolean flag = true
